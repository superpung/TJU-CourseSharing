#### 判断题
- 存在性伪造攻击对MAC（消息认证码）的破坏性比选择性伪造攻击要小
- 不可否认签名验证过程中，需要签名方参加
- 无条件安全不是现代密码学安全评估模型的类型
- DES算法不是一个群，因此可以通过重加密来提高安全强度
- CBC密码分组链接模式不需要使用解密函数
