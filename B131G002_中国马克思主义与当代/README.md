### 考试

- 形式：开卷考试，不同授课教师的题目可能不同。
- 时间：2 小时，通常在最后一周的课上进行。
- 范围：
    1. 选择感兴趣的、印象深刻的汇报主题谈谈自己的思考，可以是自己组的汇报主题，也可以是其他组的汇报主题；
    2. 每讲课程PPT结尾处的思考问题（学习通中课程-更多-资料板块课可下载）
- 要求：可携带与考试相关的纸质资料，不允许用电子设备
- 分数：两道论述题，每道50分（答题字数800字左右）
