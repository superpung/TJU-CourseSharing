# 计算机视觉

## 24252学期 孙岩标

### 简介

智感必修课，测控专业选修课。
内容为传统计算机视觉的入门版，涉及基础数学、图形学、图像处理基本方法（包括但不限于特征点提取、边缘提取）、最优估计、单/多目几何、深度学习等内容

### 平时

随机两次点名，一次5分

### 考试

全开卷期末，可带电脑

形式：期末
时间：2h

### 成绩

10%平时（两次点名，都去了就给满）+20%作业（两次）+70%期末

### 评价

作业为两次，一起交，都是比较基础的工程应用。
比如24252学期两次作业分别是二维图像识别和三维视觉计算（但是核心算法不能用OpenCV库，比如基础矩阵的计算需要自行根据原理编写算法）
老师建议C++，也可使用Java、Python

### 经验

平时好好听课就没问题，老师讲的很好
两次作业认真对待，期末考的很灵活，有设计题，但是开卷且内容如果对两次作业理解透彻会很轻松
