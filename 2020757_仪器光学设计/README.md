# 仪器光学设计

## 24251学期 张效栋

### 简介

测控仪器方向的必修，科学仪器方向的选修（别选！）
共8周，作业分老师，本班15次小组合作的小作业，涉及仿真、调研等内容。有pre，加分但不强制。
最后一大作业，一般期末周前提交。形式为个人作业，六选一。
仿真软件极其多（包括但不限于Zemax（本人挚爱）、CodeV、blender、virtuallab），经常今天用一个软件，明天换一个软件。可以和老师商量，全用一个软件做，加强对一个软件的熟练度而不是最后哪个都没学好。

### 平时

15个小作业（别的班大概3个）+4组实验

### 考试

普遍85-90

形式：大作业

### 成绩

15%平时+20%实验+65%大作业

### 评价

平时分一般都给满，实验和大作业成绩只能自求多福了
