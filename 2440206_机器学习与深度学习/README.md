# 机器学习与深度学习 

## 24252学期  王煜  曹兵  3.5学分

### 简介

由于改革，原来的《机器学习》和《深度学习》两门课程合并成了一门，主要讲述机器学习算法，例如分类，聚类，回归，集成学习，降维，线性模型，逻辑回归，决策树等，深度学习算法，例如多层感知机，全连接网络，卷积网络，RNN，注意力机制。

### 平时

包括：

机器学习的实验报告（机器学习部分的作业），35%

点名签到，2.5%

深度学习小作业，2.5%

### 期末

深度学习的大作业，60%

两个班都是没有组队，每个人都单独写

### 评价

这是AI专业的同学们正式学习AI的开始，还是可以学到一些知识的，老师讲课的水平都不错。

这门课有两个班级，由于是第一次开课，可能两边的老师没有沟通，这两个班差别比较大。各个部分的平时作业，大作业，包括占比都不一样。另一个班据说更友好，他们那边最后大作业基本上都给了98分。

### 经验

曹兵老师的作业写的代码一定要写README文档。

机器学习部分写的越多越好，我写了100多页，最后这个报告97分

深度学习部分的大作业最好整点有难度的，先多读读论文，最好不要像我这样直接实现一个不同的注意力机制就完了，这个报告我最后90分，可能就是因为我没有什么亮点。

曹兵老师讲课的水平很高，建议认真听一下。
