# 知识工程

## 24252  王鑫 张小旺

### 简介

改革之后，这门课从原来的2学分变成了3.5，与共享计划上面的 [《2440051_知识工程》](../2440051_知识工程/) 相比，课时增加了很多，实验也增加了很多。

主要介绍RDF三元组，本体建模，知识图谱等内容。

### 平时

作业+实验共占30

实验基本上就是跟着要求做即可，做的正确基本上分数在95左右，就是数量比较多，一节实验课做一个实验，最后一共16个实验。

这个实验的评分主要是看你做的对不对，只要做对了就是95分，少于这个分数就说明你有地方没按要求来。

作业只布置了一次。

### 大作业

70分

一个关于知识图谱的问答系统。

老师会给出一本书，基于这本书的一些章节，需要设计schema，抽取实体，关系，构建问答系统。

最多5人组队

### 评价

同 [《2440051_知识工程》](../2440051_知识工程/) 只能说难评，首先，最最核心的内容就是RDF三元组，你明白了这个，那这门课你就学完90%了，剩下的内容感觉没啥用，其次，最难评的就是上课，因为一方面不知道考啥，另一方面老师讲的内容又太难，有点听不懂，张小旺老师还有口音，你要特别集中精力才能听懂，王鑫老师随机点名提问，对了有加分，基本上听课比较折磨，更何况实验，大作业和上课的内容大部分没啥关联。同学们好好利用AI大模型来学习吧。

这门课AI的同学们必修，做好准备，下学期还有一门《知识工程综合实践》，也是AI必修

### 经验

上课一段时间，老师先说了期末要考试，就是看在改革，变成3.5学分的情况，然而据老师说另一个老师没出题，所以到了快期末又变成大作业，不知道下一届怎样，据老师说要真变成考试了。

大作业的情况说了要线上答辩，然后提问，不好的话全组扣分啥的。但是实际上，只有你答辩的时候没有介绍作业的必要的要求，比如作业里面说三元组要抽取3000个，你没说你们项目多少个，他会问一下，或者一些过于离谱的情况他会问一下，比如schema有几百条，或者这个项目你们用了什么方法，而这个方法那个助教感兴趣，会问一下。所以只要介绍了必要的要求，用的方法解释明白了，助教是不会提问的，这种情况的小组占了多数。并且这个提问也基本上就是普通的讨论，没有压力的情况。

关于大作业，一定要抽取足够的三元组，要求3000个，老师给每个组的分配的内容只能说不太够，所以要自己想办法，可以找其他的网站，也可以自己想办法从有限的内容里面提取更多的三元组
