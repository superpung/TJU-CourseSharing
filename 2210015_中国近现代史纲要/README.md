# 中国近现代史纲要

## 24252学期 杨丽雯

### 简介

在大一第二学期开设，主要涉及鸦片战争到二十大的中国历史。

### 平时

占比40%，有四次点名，点名是在快下课把全班挨个点一遍，期末临近周点名较为频繁；有一个期中2000字小论文写重要人物、重要会议、重要事件

### 期末

占比60%

* 形式：闭卷考试
* 时间：2小时
* 题型：40道单选题（1分\*40）+10道多选题（2分\*10，漏选超选不得分）+4道材料题（40分）

**24252期末材料题**

一、

1. 简单列举两个改良运动，并说明失败的教训。

2. 为什么说中国革命是历史的必然？

二、

1. 解放战争期间为什么共产党得到人民拥护，而国民党军队背离人心？
2. 毛泽东为什么将中国共产党回到北平比作“进京赶考”？谁是阅卷人，谁是答卷人?

三、

结合材料说明社会主义初级阶段基本路线的内容和意义

四、

1. 简述抗战精神的内涵
2. 说明抗日战争的历史意义

### 评价

杨丽雯老师重点画的非常准，基本能把90%考点划到，平时分。背诵量大概算所有思政课里最少的吧。

### 经验

客观题差不多就初中历史那个难度，基本就是考你一个历史事件有什么比较重要的原因/著作/意义/标志等，梳理清楚几个重要会议和事件基本就能考的高，不用死记硬背，客观题有印象、材料题能编就行，难度不大。
