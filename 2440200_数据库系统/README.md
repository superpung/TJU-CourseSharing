# 数据库系统

## 24252  王鑫 刘雪莉

### 简介

改革之后，这门课从成了有选修。虽然课程代码和原来的变了，但是教材、ppt、教材答案、往年题等依然可以参考本共享计划上面的 [《2440126_数据库原理》](../2440126_数据库原理/)。

### 平时

作业15分，基本是教材上面的题目。

实验25分。

与原来相比，一周上的课程更多，并且在学期中就结束，在期末周之前考试。

### 考试

60分，闭卷，2小时。

与原来相比，全英的选择题变成了基本上全部中文，但是依然存在几道英文题目。

与往年试卷有较大重合

### 评价

作业，实验给分比较友好，考试如果认真准备，那么不难。推荐选这门课。

王鑫老师上课会点名同学们回答问题，上课很有个性，会留一些额外的作业，比如翻译翻译英文的论文，自己安装mysql，然后执行一下老师给的代码等等。但是课本作业相较于其他老师少一点。

笔记参考 [《2440126_数据库原理》](../2440126_数据库原理/) 里面学长写的然后自己做题补充即可。
